# 7 Step Product Lifecycle

1. Prompt the user to create an account with Kartomo
2. Gather vehicle details from the user: make, model, year, miles, and VIN
3. Request vehicle maintenance history from the user (last oil, tire, brake-fluid change, last time car was taken in for service, etc.)
4. Aqquire/prompt the user for their location(s) where they drive
5. Provide a clean, fast, and secure interface for users to view more detailed car information (engine/componet info, online car manual, etc.
6. Continuously inform the user when any car componets or services are recommended to be completed based on their car's milage and maintenance history
7. Display a list of local spots for car repair/maintainence and cheap gas based on user location and gas preference
