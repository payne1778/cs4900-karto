# 7 Step Product Lifecycle

1. Prompt the user to create an account with Karto
2. Gather vehicle details from the user: make, model, year, miles, and VIN
3. Add vehicle maintenance history from the user (last oil, tire, brake-fluid change, last time car was taken in for service, etc.)
4. Continuously inform the user when any car services are recommended to be completed based on their maintenance history
5. Display a list of local gas stations based on user location and gas preference
6. Allow the user to select gas stations that they trust.
