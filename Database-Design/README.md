# Database Design

This directory contains all database work completed by the `Karto` group.
Below are descriptions of this directory's contents. 

### Docker

This directory contains the Docker compose file to set up the `Karto` project database with MariaDB. 
See the ![Docker directory README](./Docker/README.md) for more information about database initalization with Docker. 

### Models

This directory contains the Conceptual, Logical, and Physical Models for the `Karto` project database. 
- To learn about our Conceptual Model, see the ![Conceptual Model README](./Models/Conceptual/README.md)
- To learn about our Logical Model, see the ![Logical Model README](./Models/Logical/README.md)
- To learn about our Physical Model, see the ![Physical Model README](./Models/Physical/README.md)

### Scripts

This directory contains all SQL scripts to initalize and set up the `Karto` project database.
See the ![Scripts directory README](./Scripts/README.md) for more information about the group's database scripts. 

### 'Business Queries.md'

This Markdown file contains some common SQL business queries that the `Karto` project makes to our database.
See ![Business Queries.md](./Business%20Queries.md) to view these queries.
